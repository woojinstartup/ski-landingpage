# Ski Landing Page Template

A premium scroll-driven ski equipment landing page built with React, TypeScript,
Vite, and Tailwind CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Assets

All runtime assets are included locally under `public/assets`:

- `hero-ski-descent`: 203 JPG frames for the hero scroll sequence
- `gear-jump-sequence`: 232 JPG frames for the jump/gear scroll sequence
- `product-cards`: 3 square PNG product images

The app uses local asset paths only. No external CDN hotlinks are required.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```
